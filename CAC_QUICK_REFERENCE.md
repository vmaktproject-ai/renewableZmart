## âŒ Common Errors & Fixes

### Error: "Invalid CAC registration number format"
**Fix**: Must be RC/BN/IT/LLP followed by 6-7 digits
```
Wrong: 123456, ABC-123456, RC123
Right: RC-123456, BN-789012
```

### Error: "RC registration number is too short"
**Fix**: Must have at least 6 digits after prefix
```
Wrong: RC-123, RC-12345
Right: RC-123456, RC-1234567
```

### Error: "This appears to be a test or placeholder number"
**Fix**: Use your actual CAC registration number
```
Wrong: RC-123456 (if exactly this), RC-000000
Right: Your real CAC number from certificate
```

### Error: "Registration number appears to be invalid"
**Fix**: Don't use repeated digits
```
Wrong: RC-111111, RC-222222, RC-000000
Right: Your actual CAC number with mixed digits
```

### Error: "This business registration number is already registered"
**Fix**: This CAC number is already in use
- If it's your business, contact support
- If not, double-check your CAC certificate

## ğŸ†˜ Troubleshooting

### "I don't have my CAC certificate"
- Log into CAC portal: https://icrp.cac.gov.ng
- Request duplicate certificate
- Or visit CAC office with ID

### "My number doesn't work"
1. Check certificate for exact format
2. Remove spaces and try again: RC-123456
3. Ensure correct prefix (RC/BN/IT/LLP)
4. Count digits (should be 6 or 7)

### "System says my number is fake"
- Don't use test numbers like 123456
- Use actual number from CAC certificate
- Avoid repeated digits (111111)

### "Need to register my business first"
Visit CAC: https://www.cac.gov.ng
- Online registration available
- Get RC/BN number before vendor registration
- Costs vary by business type
